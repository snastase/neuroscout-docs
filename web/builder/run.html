After Submission#

After submitting your analysis, it will no longer be editable unless there is an error in your design that you must fix.

If your analysis successfully compiles, an example neuroscout-cli command will be shown.

For more information on how to execute an analysis, head over to the Run analysis section.

Note

After your analysis is compiled, you can change the public/private settings of your analysis, as well as edit the name and description. Also, notice the Review tab is still available.
